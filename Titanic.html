Kaggle's Titanic Competition

Introduction

According to Wikipedia:

Kaggle, a subsidiary of Google LLC, is an online community of data scientists and machine learning practitioners. Kaggle allows users to find and publish data sets, explore and build models in a web-based data-science environment, work with other data scientists and machine learning engineers, and enter competitions to solve data science challenges.

In Kaggle we can find a lot of data sets that we can use to explore, analyze, and build our models when we want to become data scientists or just for fun. One of the competitions that we can use to get started is the "Titanic - Machine Learning from Disaster" competition.

In this section, we'll explore the Titanic passengers data set and create a model to predict whether or not a passenger survived.

Data Description

We have three types of data:
• train.csv -- The training data set.
• test.csv -- The testing data set.
• gender_submission.csv -- An example of the submission.

The data variables are described below:

Variable Definition Key
Survival Survival 1=Yes, 0=No
Pclass Ticket class 1=1st, 2=2nd, 3=3rd
Sex Sex male, female
Age Age in years
SibSp Number of siblings/spouses aboard
ParCh Number of parents/children aboard
Ticket Ticket number
Fare Passenger fare
Cabin Cabin number
Embarked Port of embarkation C=Cherbourg, Q=Queenstown, S=Southampton

We can start to formulate some hypotheses about the variables that most influence passenger survival. For example, we can assume that gender and age will play an important role, due to the "women and children first" rule. Moreover, we know that the upper class tends to benefit in various situations, therefore it could be another important factor.

Another hypothesis that we can make is the correlation between variables:
• The cost of the ticket is probably related to the passenger's class and port of embarkation.
• Due to age and gender, we could possibly deduce if the passenger was accompanied or not.

It's possible that some variables, as the ticket or the cabin number, are not so relevant for the model we want to create.
